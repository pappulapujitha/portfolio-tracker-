# Portfolio Tracker

A simple and user-friendly portfolio tracker designed to organize and track personal projects, skills, and progress in one place.

## 📌 About the Project
This project helps users maintain a clear record of their work, learning journey, and achievements. It is ideal for students and beginners who want to showcase their portfolio in a structured way.

## ✨ Features
- Track personal projects
- Organize skills and learning progress
- Simple and easy-to-use design
- Beginner-friendly structure
